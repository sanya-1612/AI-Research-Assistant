# 🤖 AI Research Assistant

An AI-powered Research Assistant built using Flask and Google Gemini AI.

## Features

- AI-powered research answers
- Structured responses
- PDF report generation
- Modern UI
- Suggested research topics
- Copy answer
- Research statistics

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Google Gemini API
- ReportLab

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Open:

http://127.0.0.1:5000
